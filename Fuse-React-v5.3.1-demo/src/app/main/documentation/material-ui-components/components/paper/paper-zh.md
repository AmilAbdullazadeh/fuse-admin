# Paper 纸张组件

<p class="description">在 Material Design 中，我们在屏幕上展现了纸张的物理属性。 </p>

应用程序的的背景类似纸张一样平坦、不透明，而其行为也像纸张一样，能够改变大小、与其他纸张绑定或者调换顺序。

## 变体

如果你想要一个轮廓的曲面，你可以使用 `variant` 属性。
